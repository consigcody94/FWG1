## Technical Information

### System Architecture

- **CMS:** Sveltia CMS (Git-based)
- **Frontend:** Next.js 16 with App Router
- **Hosting:** Netlify
- **Repository:** GitHub
- **Content Format:** Markdown files with YAML frontmatter
- **Authentication:** GitHub OAuth

### Publishing Workflow

```
Edit in CMS ‚Üí Save ‚Üí GitHub Commit ‚Üí Netlify Build ‚Üí Live Website
              (instant)  (instant)      (2-3 min)     (live!)
```

### Access Control

- Access controlled via GitHub repository collaborators
- Must have access to `consigcody94/FWG1` repository
- Add users at: https://github.com/consigcody94/FWG1/settings/access
