# Image Assets Guide

## Required Images for Production

This website is **fully functional** without images, but adding these will enhance the professional appearance:

### Critical (Branding)
1. **Company Logo** - `fwg-logo.png`
   - Transparent PNG
   - Recommended: 500x200px
   - Used in: Navigation, footer

2. **Favicon Set**
   - `favicon-16x16.png` (16x16px)
   - `favicon-32x32.png` (32x32px)
   - `apple-touch-icon.png` (180x180px)
   - `favicon.ico` (ICO format)

3. **Social Media Preview** - `fwg-social-preview.jpg`
   - 1200x630px
   - Shows when shared on Facebook/LinkedIn/Twitter
   - Should include: Logo, tagline, professional background

### Optional (Enhanced Experience)
- Team member photos
- Service illustrations
- Client logos (with permission)
- Project screenshots
- Office photos
- Hero background image

## How to Get Images

### Option 1: Professional Design
- Hire a designer on Fiverr/Upwork
- Cost: $50-200
- Turnaround: 2-5 days

### Option 2: Use Stock Photos
Free sources:
- Unsplash.com (high-quality, free)
- Pexels.com (free stock photos)
- Pixabay.com (free images)

Paid sources:
- Shutterstock.com
- iStock.com
- Adobe Stock

### Option 3: AI Generation
- DALL-E (OpenAI)
- Midjourney
- Stable Diffusion

### Option 4: DIY Tools
- Canva.com (free logo maker)
- Figma.com (design tool)
- Favicon.io (favicon generator)

## Image Specifications

### Logo
- **Format:** PNG with transparency
- **Size:** 500x200px (or proportional)
- **Colors:** Match brand colors (primary: #0056b3)
- **Background:** Transparent

### Favicons
Use a favicon generator:
1. Go to https://realfavicongenerator.net/
2. Upload your logo
3. Download all sizes
4. Copy to root directory

### Social Preview
- **Size:** Exactly 1200x630px
- **Format:** JPG or PNG
- **Content:** Logo + tagline + professional background
- **Text:** Large enough to read when small
- **Example:** https://placekitten.com/1200/630
